# Deepseek Decompile Plugin

## Описание

Плагин **Deepseek Decompile** предназначен для использования с Hex-Rays Decompiler. Он предоставляет несколько полезных функций для анализа и модификации декомпилированного кода, а также взаимодействует с Deepseek API для выполнения различных задач.

## Основные функции

1. **Декомпиляция текущей функции**:
   Плагин позволяет декомпилировать текущую функцию и выводить её в окно сообщений.

2. **Поиск и вывод вызовов функций**:
   Функция `find_and_print_calls` находит и выводит все вызовы функций внутри текущей функции.

3. **Переименование локальных переменных**:
   Функция `rename_lvar2` позволяет переименовывать локальные переменные в функции.

4. **Переименование функций**:
   - `rename_specific_function` переименовывает конкретную функцию.
   - `rename_all_functions` переименовывает все функции, найденные в текущей функции.

5. **Анализ элементов функции**:
   Функция `analyze_function_elements` анализирует локальные переменные и глобальные переменные, используемые в функции.

6. **Отправка запроса в Deepseek API**:
   Функция `SendRequestToDeepseek` отправляет POST-запрос к Deepseek API с заданным контентом и выводит ответ.

## Использование

Для использования плагина, поместите курсор внутри функции и запустите плагин. Плагин выполнит декомпиляцию текущей функции, проанализирует её элементы, переименует функции и переменные, а также отправит запрос к Deepseek API.
